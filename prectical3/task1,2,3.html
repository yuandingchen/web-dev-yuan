Research for and describe the ways to open developers console in Mozilla Firefox and Safari.

Here's how to open the developer console in Mozilla Firefox:

Open the Firefox browser. Press the F12 key on your keyboard or select Developer Tools from the browser menu. In the side panel that pops up, select the "Console" tab. Enter the JavaScript code in the console input box and press Enter to run it.

Here's how to open the developer console in Apple's Safari browser:

Click the menu icon in the upper right corner of Safari. In the pop-up submenu, click "Preferences". In the settings window, click on the "Advanced" option. The "Develop" menu option will be displayed below, click to check it. On the menu bar, click the Develop menu item and select Show Error Console.

What way of including internal links into the HTML page you would consider generally preferable: absolute or relative links? Describe use cases for choosing absolute links and choosing relative links.

Preference is given to relative links

Flexibility: Relative links adapt to changes in page structure and are not affected by domain name or protocol changes.

Simplicity: Relative links are generally shorter than absolute links, making the code easier to read and maintain.

Use cases for relative links

  • Links to other pages within the same website.
  • Links need to be adapted to different environments (development, testing, production).

Use cases for absolute links:

  • Links to pages or resources on other websites.
  • Need to ensure that the link is not affected by relative path changes.

Research and write about image formats used in web-development. What is the difference between JPG, GIF and PNG formats? What is WEBP image type? What is SVG?

JPG (JPEG): Lossy compression: Part of the image information is lost during the compression process, so the file size is smaller.

PNG: Lossless compression: Larger file size without loss of image quality.

GIF: Lossless compression: suitable for simple images or animation effects.

WEBP: A new image format developed by Google. At the same resolution, the size is more than 25% smaller than JPG and PNG. Supports lossy and lossless compression, suitable for still images and animations

SVG (Scalable Vector Graphics): Vector graphics format that uses XML to describe images. It can be enlarged without distortion and is suitable for drawing corporate logos, icons, etc. Unlike images, SVG is text-based and can be easily modified.
